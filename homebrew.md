# Homebrew

Homebrew 是用在 macOS 上的系统包管理工具。

[https://brew.sh/](https://brew.sh/)

## 安装

打开系统自带的终端工具，然后执行：

```
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

**brew**

安装成功以后，就可以使用 `brew` 命令去管理在 macOS 平台上的系统软件包了。

```
brew --help
```
