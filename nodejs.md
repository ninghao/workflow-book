# Node.js

安装了 Node.js 的机器就可以运行 JavaScript 语言了。 也就是有了 Node.js 以后，你可以使用 JavaScript 语言去开发服务端的应用程序。

即使你不打算学习 Node.js，平时的开发学习与工作，你的电脑上也需要安装一个 Node.js。因为很多开发时用的工具都需要用到 Node.js 。

[https://nodejs.org](https://nodejs.org)
